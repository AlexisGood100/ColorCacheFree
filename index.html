Description

Welcome to the Color Cache Game.

Today you will be sorting colors to the corresponding color cache.

Click a color and then click the pallet that has the same color (use number if unsure) to score points.

If the color is placed in the wrong color cahce, the result will be a misload which will lead to a reduction in points.

You can stack up to three colors but this will cause fatigue to increase quicker.

To increase stamina you must drink water.

To decrease fatigue you must not select any colors for two seconds.

You can stack colors but if the weight exceeds 150 then the color will not be able to join the stack.

The weight of the colors will increase fatigue while the actual handling of the color will decrease stamina.
